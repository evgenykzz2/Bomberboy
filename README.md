# Bomberboy
Bomberman port for Arduboy

![Demo](Images/demo.gif)

Features:
- Original music
- Up to 16 bombs
- Up to 24 blocks explosion radius
- Up to 16 enemies
- 50 Levels
- 8 Bonuses
- 8 Enemies types
- The ending of the game is present
